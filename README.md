## ğŸ—ï¸ Key Features & Architecture
[cite_start]The system is built using a modern microservices architecture, ensuring that the frontend and backend can scale independently[cite: 37].

* [cite_start]**Containerization:** Both frontend and backend services are packaged using **Docker** for consistent behavior across all environments[cite: 38].
* [cite_start]**Serverless Deployment:** The application is orchestrated by **AWS ECS Fargate**, removing the need to manage EC2 instances[cite: 39].
* [cite_start]**Scalable Database:** Uses **Amazon DynamoDB** as a serverless key-value store for menu data and orders, ensuring single-digit millisecond latency[cite: 40].
* [cite_start]**Automated CI/CD:** A custom pipeline built with **PowerShell** and **AWS CLI** automates the build-and-deploy lifecycle to Amazon ECR[cite: 11, 41].
